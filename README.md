# TestFightGame —— 文字格斗游戏

一款基于 Java 控制台的文字格斗游戏，玩家通过自由分配属性点创建角色，与不同类型的敌人进行回合制战斗。战斗中可拾取和使用道具恢复生命值，挑战越来越强的对手，争取更高的胜场记录。

## 游戏特色

- **自由属性分配** —— 30 点属性自由分配到生命值、攻击力、防御力、蓝条，打造独一无二的角色
- **回合制战斗** —— 玩家先手，策略性选择技能或使用道具击败敌人
- **道具系统** —— 战斗胜利后有 30% 概率掉落回血道具，可在战斗中使用
- **多样敌人** —— 4 种不同风格的敌人（战士、刺客、坦克、法师），各有独特技能
- **难度递增** —— 每次胜利后敌人变强，挑战你的极限
- **属性成长** —— 每累计 3 场胜利，玩家获得额外属性提升
- **登录系统** —— 包含用户注册/登录功能，支持验证码验证和账号安全机制

## 如何运行

### 方式一：使用 IntelliJ IDEA（推荐）

1. 用 IntelliJ IDEA 打开项目文件夹
2. 等待项目索引完成
3. 打开 `src/App.java`
4. 右键点击 `Run 'App.main()'` 或使用快捷键 `Shift + F10`

### 方式二：使用 VS Code

1. 用 VS Code 打开项目文件夹
2. 安装 [Extension Pack for Java](https://marketplace.visualstudio.com/items?itemName=vscjava.vscode-java-pack) 扩展
3. 打开 `src/App.java`，点击 `Run` 按钮

### 方式三：命令行编译运行

```bash
# 进入项目目录
cd TestFightGame

# 编译所有 Java 文件
javac -d out src/App.java src/com/itmuzee/domain/*.java src/com/itmuzee/ui/*.java

# 运行程序
java -cp out App
```

## 游戏玩法详解

### 角色创建

进入游戏后，你将拥有 **30 个自由属性点**，可以分配到以下四个维度：

| 属性 | 每点效果 | 基础值 | 说明 |
|------|---------|--------|------|
| **生命值 (HP)** | +10 HP | 100 | 决定你能承受多少伤害 |
| **攻击力 (Attack)** | +2 攻击 | 10 | 决定你造成伤害的基础值 |
| **防御力 (Defense)** | +1 防御 | 0 | 减少受到的伤害 |
| **蓝条 (MP)** | +5 MP | 0 | 用于释放技能的资源 |

**策略建议**：
- 🗡️ **攻击型**：高攻击力，快速击杀敌人，但容错率低
- 🛡️ **防御型**：高防御力，减少受到的伤害，适合持久战
- 🔮 **技能型**：高蓝条，频繁使用技能
- ❤️ **平衡型**：均衡分配，适应各种敌人

### 战斗系统

游戏采用 **回合制战斗**，流程如下：

```
┌─────────────────────────────────────────────────┐
│                  战斗开始                         │
│                    ↓                             │
│            显示双方状态和血条                      │
│                    ↓                             │
│    ┌──→ 玩家回合：选择技能                        │
│    │        ↓                                    │
│    │   计算伤害 → 显示伤害结果                     │
│    │        ↓                                    │
│    │   敌人死亡？──→ 是 ──→ 战斗胜利              │
│    │        │              ↓                     │
│    │        否         恢复生命值                  │
│    │        ↓              ↓                     │
│    │   敌人回合：AI 随机行动      下一场战斗        │
│    │        ↓                                    │
│    │   计算伤害 → 显示伤害结果                     │
│    │        ↓                                    │
│    │   玩家死亡？──→ 是 ──→ 游戏结束              │
│    │        │                                    │
│    │        否                                    │
│    └────────┘                                    │
└─────────────────────────────────────────────────┘
```

**伤害计算公式**：
```
实际伤害 = 攻击方攻击力 - 防御方防御力
最低伤害 = 1（保证至少造成 1 点伤害）
```

**血条显示**：
游戏使用 20 格进度条直观展示双方生命值状态：
```
张三 HP: 85/100 [████████████████░░░░] 85%
敌人 HP: 60/80  [███████████████░░░░░] 75%
```

### 技能介绍

玩家拥有 **3 个技能**，每个技能有不同的效果和代价：

| 技能 | 效果 | 代价 | 使用条件 |
|------|------|------|---------|
| **普通攻击** | 造成 `攻击力 - 敌方防御力` 点伤害 | 无 | 无限制 |
| **强力一击** | 造成 `攻击力 × 1.8 - 敌方防御力` 点伤害 | 消耗 10 HP | HP ≥ 10 |
| **生命汲取** | 随机恢复 20~40 HP | 消耗 1 MP | MP ≥ 1 |
| **使用道具** | 消耗一个背包中的道具，恢复对应生命值 | 消耗道具 | 背包中有道具 |

**技能策略**：
- 普通攻击是最稳定的选择，无副作用
- 强力一击适合需要快速击杀敌人时使用，但注意自身血量
- 生命汲取是续航技能，消耗蓝量换取生命恢复
- 使用道具是紧急回血手段，道具来源于战斗掉落

### 道具系统

战斗胜利后有 **30% 概率** 随机获得一个回血道具，道具会存入玩家背包中，可在战斗中随时使用。

**道具掉落池**：

| 道具名称 | 回复量 |
|---------|--------|
| 🍑 桃子 | +10 HP |
| 🍳 煎蛋 | +20 HP |
| 🍗 花酿鸡 | +30 HP |
| 🐟 黑背鲈鱼 | +40 HP |
| 🍲 白玉汤 | +50 HP |

> 💡 使用道具会消耗该道具，同种道具可叠加存放于背包中。

### 敌人类型

游戏中有 **4 种敌人**，每种都有独特的战斗风格：

| 敌人 | 生命值 | 攻击力 | 防御力 | 技能 | 技能效果 |
|------|--------|--------|--------|------|---------|
| 🗡️ **初级战士** | 80 | 15 | 10 | 猛击 | 造成 1.5 倍攻击伤害 |
| 🗡️ **敏捷刺客** | 60 | 20 | 5 | 快速攻击 | 造成 2 次 0.5 倍攻击伤害 |
| 🛡️ **重装坦克** | 120 | 10 | 20 | 防御姿态 | 下次受伤减半 |
| 🔮 **神秘法师** | 70 | 25 | 8 | 火球术 | 造成 1.8 倍攻击伤害 |

**敌人 AI**：
- 每回合有 50% 的概率使用技能，50% 的概率普通攻击
- 每场战斗随机选择一个敌人作为对手

### 难度递进

游戏具有 **动态难度系统**，随着你的胜利，挑战会越来越大：

**敌人成长**（每次玩家胜利后）：
- 生命值上限 +10
- 攻击力 +3
- 防御力 +2

**玩家成长**（每累计 3 场胜利）：
- 生命值上限 +30
- 攻击力 +5
- 防御力 +3

**战斗恢复**：
- 每场战斗结束后，玩家随机恢复 20~40 点生命值
- 每场战斗结束后，玩家随机恢复 5~9 点蓝量
- 战斗结束后有 30% 概率掉落随机回血道具

## 技术实现

### 核心设计模式

- **面向对象继承**：`Characters` 作为基类，`HeroCharacters` 和 `EnemyCharacters` 继承并扩展
- **方法重写**：`EnemyCharacters` 重写 `takeDemage()` 方法实现防御姿态的伤害减半机制
- **封装性**：属性私有化，通过 getter/setter 访问

### 类图关系

```
Characters (角色基类)
├── name: String
├── HP: int
├── maxHP: int
├── attack: int
├── defense: int
├── isAlive(): boolean
├── heal(amount): void
├── takeDemage(demage): void
└── show(): String
    │
    ├── HeroCharacters (玩家英雄)
    │   ├── skillList: ArrayList<String>
    │   ├── inventory: ArrayList<Consumable>
    │   ├── MP: int
    │   ├── maxMP: int
    │   ├── showSkill(): String
    │   ├── useMP(amount): void
    │   ├── restoreMP(amount): void
    │   ├── addConsumable(c): void
    │   ├── useConsumable(name): void
    │   └── showConsumables(): void
    │
    └── EnemyCharacters (敌方角色)
        ├── skill: String
        ├── defending: boolean
        └── takeDemage(demage): void  [重写：防御减伤]

Consumable (消耗品)
├── name: String
├── healAmount: int
└── toString(): String

User (用户实体)
├── id: String
├── username: String
├── password: String
└── status: boolean
```
